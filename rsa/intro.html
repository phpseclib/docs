Supported Public Key Formats

  • PKCS#1
  • OpenSSH
  • XML Signature

Features

  • Create Key Pairs
  • Convert keys to different formats
  • Create / verify signatures
    • Using the (more common) PKCS#1 scheme
    • Using the (more secure) Probabilistic Signature Scheme
  • Encrypt / decrypt strings
    • Using the (more common) PKCS#1 scheme
    • Using the (more secure) Optimal Asymmetric Encryption Padding (more info)
  • Utilizes RSA Blinding
