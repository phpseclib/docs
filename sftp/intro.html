SFTP extends SSH2

SFTP has all the functions SSH2 has and then some.

Features

  • Upload Files (put)
    • Upload from a string or a local file
    • Resume uploads
  • Download Files (get)
    • Download to a string or to the filesystem
  • Directory Management
    • Change the directory (chdir)
    • Create a directory (mkdir)
    • Delete a directory (rmdir or delete)
    • Return the current directory (pwd)
  • Directory Listing (nlist and rawlist)
  • Change Attributes
    • Permissions (chmod)
    • Date (touch)
    • (Group) Ownership (chgrp, chown)
    • Size (truncate)
  • File Information (stat, lstat and size)
  • Delete and Rename (delete and rename)
  • Logging
